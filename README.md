# Landing Page

Simple landing page project containing a single static page built with HTML, CSS and JavaScript.

## Files
- `landing.html` — the main HTML file (open in a browser).
- `style.css` — styles for the page.
- `script.js` — optional JS for interactivity.

## Quick start
1. Clone the repository or download the files.
2. Open `landing.html` in your browser.
